# Getting Started

CarSim を使い始めるための 4 ステップです。順番に進めてください。

## ステップチェックリスト

- [ ] **ステップ 1 — インストール**
  CarSim をコンピュータにインストールします。
  → [インストールガイドへ](./install)

- [ ] **ステップ 2 — ライセンス設定**
  ライセンスを認証してソフトウェアを有効化します。
  → [ライセンス設定へ](./license)

- [ ] **ステップ 3 — クイックスタート**
  初めてのシミュレーションを実行します。
  → [クイックスタートへ](./quick-start)

- [ ] **ステップ 4 — トレーニング準備**
  月次トレーニングに向けた準備を行います。
  → [トレーニング準備へ](./training-prep)

## 困ったときは

手順で詰まった場合は [FAQ](/kb/faq/) または [サポート](/support/) をご参照ください。
