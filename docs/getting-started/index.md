# はじめる

CarSim を使い始めるための 3 ステップです。順番に進めてください。

## ステップチェックリスト

- **ステップ 1 — インストール**
  CarSim をコンピュータにインストールします。
  → [インストールガイドへ](./install)

- **ステップ 2 — ライセンス設定**
  ライセンスを認証してソフトウェアを有効化します。
  → [ライセンス設定へ](./license)

- **ステップ 3 — クイックスタート**
  初めてのシミュレーションを実行します。
  → [クイックスタートへ](./quick-start)

## トレーニングについて

Applied Intuition Japan では定期的に CarSim トレーニングを開催しています。
申し込み済みの方は、担当者からご案内した資料と準備手順をご確認ください。

→ [トレーニング準備ページ](./training-prep)
