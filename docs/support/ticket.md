# チケット提出ガイド

技術的な問題は [CarSim.com のサポートポータル](https://www.carsim.com/support/) からチケットを提出してください。

## 効果的なチケットの書き方

迅速な対応のために、以下の情報を含めてください。

### 必須情報

| 項目 | 内容の例 |
|------|---------|
| **CarSim バージョン** | `CarSim 2024.0` |
| **OS / バージョン** | `Windows 11 Pro 23H2` |
| **ライセンスタイプ** | `ネットワークライセンス` |
| **問題の発生タイミング** | `シミュレーション実行時` |

### 問題の説明

1. **何をしようとしていたか** — 操作手順を具体的に
2. **何が起きたか** — エラーメッセージ全文をコピー＆ペースト
3. **期待した動作** — 本来どうなるべきか
4. **再現手順** — 問題を再現できる最小限の手順

### 添付すると解決が早まるもの

- スクリーンショット（エラーダイアログ全体が写っているもの）
- ログファイル（`Documents\CarSim_N\Simfile\` 以下の `.log` ファイル）
- 問題が発生したシムファイル（`.cpar`）
