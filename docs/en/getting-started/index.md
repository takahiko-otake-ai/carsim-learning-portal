# Getting Started

Follow these 3 steps in order to begin using CarSim.

## Step Checklist

- **Step 1 — Install**
  Install CarSim on your computer.
  → [Go to Installation Guide](./install)

- **Step 2 — License Setup**
  Activate your license to enable the software.
  → [Go to License Setup](./license)

- **Step 3 — Quick Start**
  Run your first simulation.
  → [Go to Quick Start](./quick-start)

## Training
